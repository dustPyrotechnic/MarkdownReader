# Lists And Quotes

## Unordered

- 第一项
- 第二项
  - 第二项的子项 A
  - 第二项的子项 B
    - 第三级子项
- 第三项

## Ordered

1. 第一项
2. 第二项
   1. 子项 1
   2. 子项 2
3. 第三项

## Mixed Markers

1. 有序项
   - 无序子项
   - 另一个无序子项
2. 第二个有序项

## Loose List

- 第一项第一段。

  第一项第二段，应该仍属于第一项。

- 第二项。

## Task List

- [ ] 未完成任务
- [x] 已完成任务
- [X] 大写 X 是否识别取决于实现
- [ ] 包含 **格式** 和 `code` 的任务

## Ambiguous Lines

-列表标记后没有空格，通常不是列表
1.这是数字后没有空格，通常不是列表

## Blockquotes

> 一级引用。
>
> > 二级引用。
> >
> > - 引用里的列表
> > - 第二项
>
> 回到一级引用。

普通段落，不应该继续在引用里。

> 引用中包含代码：
>
> ```swift
> let message = "Hello from quote"
> ```

> 引用最后一行后面没有空行
下一行是否仍在引用中，取决于解析规则；CommonMark 中这里通常仍是引用延续。
